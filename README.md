# Otto-bot

## Project Description
Otto-bot is a Slack bot for the Ottoneu Fantasy Baseball community that performs helpful tasks when discussing players.

## Features
* Provides ability to link to players' Ottoneu, FanGraphs, and/or StatCast player pages
* Provides ability to create at trade review post that gathers salary and loan information automatically and posts a consistent format with pre-voting.

## Architecture
Otto-bot is set up to run as AWS lambda functions against a MongoDB data-storage container.
